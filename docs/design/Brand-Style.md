## Themes

The UMTAS platform supports multiple visual themes for different user contexts.

### Dark Mode (Default)

Our primary interface uses a high-contrast charcoal and white palette, optimized for professional developer and administrative environments.

### Light Mode

A clean, minimalist white theme with soft grey accents for standard university administrative portal parity.
